`Next.js` `TypeScript` `Python` `FastAPI` `PostgreSQL` `Cloudflare R2` `Docker`

### 📸 [Lamella](https://github.com/Duarte0) — Premium Photo Delivery SaaS

`React` `TypeScript` `Vite` `Supabase` `Cloudflare R2` `Stripe` `Framer Motion`
